Arcii's Discord Colored ANSI Art Generator

About

This is a simple app that creates colored Discord art using the ANSI color codes available on the latest Discord desktop versions.

To use this, select color, choose boxes, then copy it using the button below, and send in a Discord message.

Last color is an eraser.
